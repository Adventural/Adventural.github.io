正则表达式

re.escape(pattern)

可以对字符串中所有可能被解释为正则运算符的字符进行转义的应用函数。如果字符串很长且包含很多特殊技字符，而你又不想输入一大堆反斜杠，或者字符串来自于用户(比如通过raw_input函数获取输入的内容)，且要用作正则表达式的一部分的时候，可以使用这个函数。

Class内部定义变量

带参数的super和不带参数的super

Python2和Python3不同的语法。Python3允许使用不带参数的super函数，而Python2的super函数中需要带有参数。

如果是Python2的话，需要写成super(C, self).method(arg)

*args和**kwargs：python支持的可变参数

*args

接收可变个非关键字参数，如test(x, y, "abc")，存放在元组中

可变参数一定要在普通参数之后，如果调换顺序会报错。

**kwargs

接收可变个关键字参数，如testFun(x=1, y=2)，存放在字典中

callable函数和__call__方法

__call__方法

实现了__call__的类，其实例化对象可以通过函数的方式调用

callable函数

用于检查一个对象是否是可调用的。如果返回 True，object 仍然可能调用失败；但如果返回 False，调用对象 object 绝对不会成功。

对于函数、方法、lambda 函式、 类以及实现了__call__ 方法的类实例，它都返回 True。

typing类型提示支持

详细的类型提示支持可以从官方描述文档获取typing — 类型提示支持 — Python 3.10.0 文档

下面介绍一下常用的类型提示：

Tuple

解释器会提示为Tuple类型

Union

和单词的意思一样，代表变量的数据类型可以是多个中的一个，如函数有可能根据不同情况有时返回str或返回list，那么就可以写成Union[list, str]

Optional

Optional 是Union的一个简化， 当 数据类型中有可能是None时，比如有可能是str也有可能是None，则Optional[str], 相当于Union[str, None]。

NewType

自定义类型，可以参考官方描述文档的用法

Callable

Callable[[Arg1Type, Arg2Type], ReturnType]通常实现回调函数的类型提示

也可以使用简单的…来替换类型提示的参数列表Callable[..., ReturnType]

IOBase类内定义的read和write，是通过Callable来声明的

继承IOBase抽象类的类需要实现read和write方法，否则直接调用会出错

Any

Any是一种特殊的类型，如它本身的意思一样它可以是所有的类型。

静态类型检查器认为所有类型均与 Any兼容，同样Any 也与所有类型兼容。Any类型在赋值时不做类型检查，即便将一个实际为int类型的Any类型赋值给str类型。
